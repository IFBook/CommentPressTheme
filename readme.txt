== Description ==

Commentpress allows readers to comment paragraph by paragraph in the margins of a text. You can use it to annotate, gloss, workshop, debate and more!

== Installation ==

This theme only works in combination with the Commentpress Plugin:
https://github.com/IFBook/CommentPressPlugin
Please install and activate the theme by doing:

1) Unzip the .zip file into `/wp-content/themes/commentpress`
2) Activate the theme
3) Visit settings pages and configure theme
3) You are done!
